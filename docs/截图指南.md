## 🛠️ 截图工具推荐

### macOS
- **系统自带**：Cmd + Shift + 4
- **CleanShot X**：专业截图工具
- **Shottr**：免费轻量

### Windows
- **系统自带**：Win + Shift + S
- **Snipaste**：免费强大
- **Greenshot**：开源

### Linux
- **Flameshot**：功能强大
- **Shutter**：老牌工具
- **系统自带**：PrintScreen

## 📸 截图步骤

### 1. 准备环境

```bash
# 启动项目
cd linguamate-pro
bash scripts/quick-start.sh
```

### 2. 打开浏览器

访问 http://localhost:3000

### 3. 进入演示模式

点击各页面的"快速演示"按钮

### 4. 调整窗口大小

推荐窗口尺寸：1920x1080（全屏）

### 5. 开始截图

使用上述工具截图，保存到 `docs/screenshots/` 目录
