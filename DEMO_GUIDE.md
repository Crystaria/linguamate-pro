# 🎬 LinguaMate Pro 演示指南

> 本文档用于指导如何在面试/演示场景中展示 LinguaMate Pro 项目

---

## 📋 演示前准备（2 分钟）

### 1. 启动项目

```bash
# 方式 1：一键启动（推荐）
cd linguamate-pro
bash scripts/quick-start.sh

# 方式 2：分别启动
# 后端
cd backend && python main_local.py
# 前端（新终端）
cd frontend && npm start
```

### 2. 打开浏览器

访问：http://localhost:3000

### 3. 准备演示数据

系统已预设演示数据，点击"**快速演示**"按钮即可自动填充。

---

## 🎯 演示流程（5-8 分钟）

### 第一部分：项目介绍（1 分钟）

**话术示例：**
> "这是我独立开发的 AI 语言学习平台 LinguaMate Pro，融合了语言学理论和 AI 技术，提供个性化的语言学习体验。项目采用 React + FastAPI 全栈架构，支持文本分析、图片识别和 AI 对话三大核心功能。"

**展示内容：**
- 首页界面设计
- 技术栈介绍（React + FastAPI + Tailwind CSS）
- 多语言支持（中英文切换）

---

### 第二部分：核心功能演示（4-5 分钟）

#### 功能 1：智能文本分析（1.5 分钟）

**操作步骤：**
1. 点击"文本分析"
2. 点击"快速演示"（自动填充示例文本）
3. 选择学习等级（初级/中级/高级）
4. 点击"分析"

**展示亮点：**
- ✅ 词汇分析：词根词缀解析
- ✅ 语法结构：句子类型分析
- ✅ 语义理解：主题识别
- ✅ 练习生成：自动生成分级练习题

**话术示例：**
> "这里我设计了分级学习系统，根据用户选择的等级，AI 会调整分析深度和练习题难度。比如高级模式下，会增加批判性思维和学术表达的分析。"

---

#### 功能 2：图片内容识别（1.5 分钟）

**操作步骤：**
1. 点击"图片分析"
2. 上传示例图片（或点击"使用示例图片"）
3. 等待 OCR 识别和分析

**展示亮点：**
- ✅ OCR 文字识别（中英文支持）
- ✅ 图片内容语言学分析
- ✅ 基于图片内容生成练习题

**话术示例：**
> "这个功能使用了 Tesseract OCR 进行文字识别，然后送入 AI 模型进行语言学分析。用户可以上传教材截图、文章图片等，系统会自动识别并分析。"

---

#### 功能 3：AI 对话练习（2 分钟）

**操作步骤：**
1. 点击"对话练习"
2. 选择预设场景（如"餐厅点餐"）
3. 开始对话演示
4. 展示上下文记忆功能

**展示亮点：**
- ✅ 多场景支持（餐厅/购物/问路等）
- ✅ 上下文记忆（AI 记住对话历史）
- ✅ 分级对话（难度可调）
- ✅ 自定义场景（用户可创建）

**话术示例：**
> "对话系统支持上下文记忆，AI 会记住之前的对话内容，保持连贯性。我还设计了自定义场景功能，用户可以创建个性化的对话场景。"

---

## 📊 技术问答准备

### 可能被问到的问题：

| 问题 | 回答要点 |
|------|----------|
| 为什么选 React？ | 生态丰富，组件化开发，适合快速迭代 |
| 为什么选 FastAPI？ | 性能好，自动生成 API 文档，类型安全 |
| 如何处理并发？ | 后端使用异步处理，前端使用乐观更新 |
| 数据如何存储？ | 学习记录使用本地存储，可扩展到数据库 |
| 如何保证代码质量？ | 单元测试 + ESLint + Prettier |
| 如何部署？ | 前端 Vercel，后端 Railway，一键部署脚本 |
