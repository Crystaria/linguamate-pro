## 🎯 第一步：部署前端到 Vercel（2 分钟）

### 1. 打开 Vercel
```
https://vercel.com/new
```

### 2. 用 GitHub 登录
- 点击 "Continue with GitHub"
- 授权 Vercel 访问你的 GitHub

### 3. 导入项目
- 找到 `Crystaria/linguamate-pro`
- 点击 "Import"
- **Root Directory** 填：`frontend`
- 点击 "Deploy"

### 4. 等待部署完成
- 约 2-3 分钟
- 成功后会显示：`https://linguamate-pro-xxx.vercel.app`
- **📋 复制这个链接，后面要用**

---

## 🔧 第二步：部署后端到 Railway（2 分钟）

### 1. 打开 Railway
```
https://railway.app/new
```

### 2. 用 GitHub 登录
- 点击 "Login with GitHub"
- 授权 Railway 访问你的 GitHub

### 3. 创建新项目
- 点击 "New Project"
- 选择 "Deploy from GitHub repo"
- 找到 `Crystaria/linguamate-pro`
- 点击 "Deploy Now"

### 4. 配置后端
- 点击刚创建的项目
- 点击 "Settings"
- **Root Directory** 填：`backend`
- **Start Command** 填：`python main_local.py`

### 5. 添加环境变量
在 Railway 项目页面，点击 "Variables"，添加：
```
PORT=8000
HOST=0.0.0.0
DEBUG=False
AI_MODE=local
```

### 6. 等待部署完成
- 约 3-5 分钟
- 成功后点击 "Settings" → "Domains"
- **📋 复制这个链接，例如：`https://xxx.railway.app`**

---

## 🔗 第三步：连接前后端（1 分钟）

### 1. 回到 Vercel
- 打开你的 Vercel 项目页面
- 点击 "Settings" → "Environment Variables"

### 2. 添加环境变量
- 点击 "Add New"
- **Name**: `REACT_APP_API_URL`
- **Value**: 刚才复制的 Railway 链接（例如 `https://xxx.railway.app`）
- 勾选 "Production" 和 "Preview"
- 点击 "Save"

### 3. 重新部署
- 点击 "Deployments"
- 找到最新的部署
- 点击 "..." → "Redeploy"
- 等待 1-2 分钟

## 🆘 遇到问题？

### Vercel 部署失败
- 检查是否正确设置 Root Directory 为 `frontend`
- 查看 Vercel 的部署日志

### Railway 部署失败
- 检查 Start Command 是否为 `python main_local.py`
- 查看 Railway 的部署日志

### 前端无法连接后端
- 检查 Vercel 环境变量是否正确
- 检查 Railway 服务是否正常运行
- 可能需要等待 1-2 分钟让服务启动
